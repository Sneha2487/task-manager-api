# Task-manager-api
A backend Task Management REST API built using Java and Spring Boot, enabling users to create, update, delete, and track tasks with database persistence and clean layered architecture.

---

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- REST APIs

---

## Features
- Create, update, and delete tasks
- Track task status (Pending, Completed)
- Due date support
- Input validation
- Clean layered architecture

---
